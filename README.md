# Exam Trainer

iOS 프로그래밍과 웹프레임워크2 시험 대비 md 노트를 기반으로 만든 React + TypeScript + Tailwind 문제 풀이 사이트입니다.

## 풀이 항목

- iOS 전체 풀기
- 웹프레임워크2 전체 풀기
- iOS 빠른풀기 20제
- 웹프레임워크2 빠른풀기 20제

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

Vercel에서는 `vercel.json` 기준으로 `npm run build` 후 `dist`를 배포합니다.
